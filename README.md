Програма "Гра 'Вгадай число'" є простою інтерактивною грою, де користувач повинен вгадати випадкове число в заданому діапазоні. Основна мета гри - знайти загадане число за якнайменшу кількість спроб.



Основні можливості програми:


Вказання діапазону чисел: Користувач може вказати діапазон чисел, в якому буде згенеровано випадкове число для вгадування.


Вгадування числа: Користувач вводить свої варіанти чисел і отримує відповідь про те, чи воно більше чи менше від загаданого числа.


Виведення результату: Гра завершується, коли користувач вгадав число. Виводиться вікно з повідомленням про виграш і кількість спроб, яку зробив гравець.


Нова гра: Користувач може почати нову гру, вказавши новий діапазон чисел, не закриваючи вікно програми.



Ця програма використовує бібліотеку PySimpleGUI для створення простого графічного інтерфейсу. Кожна гра генерує нове випадкове число, і користувач може продовжувати грати, поки не бажає завершити програму.
